# 链上监控清单

DeFi 协议上线后必须监控的关键指标。

## 实时告警

| 事件 | 阈值 | 严重程度 |
|------|------|---------|
| 异常大额转账 | > \$100K 等效 | Critical |
| 预言机价格偏离 | > 5% 偏离 Chainlink | High |
| 存款/借款异常 | 超过 TVL 20% | High |
| 治理投票异常 | 单一地址 > 30% 票权 | High |
| 合约升级 | 时间锁之外的操作 | Critical |

## 钱包监控

### 管理员/多签钱包
- [ ] 监控所有 Admin 钱包的链上活动
- [ ] 多签钱包的签名阈值变化
- [ ] timelock 执行窗口

### 大户钱包
- [ ] 监控鲸鱼地址的存款/提款模式
- [ ] 监控关联地址之间的资金转移

## 工具推荐

| 工具 | 用途 |
|------|------|
| Forta Network | 去中心化实时监控 |
| OpenZeppelin Defender | Admin + 自动化响应 |
| Tenderly | 交易模拟 + 实时监控 |
| Dune Analytics | 链上数据可视化 |

## 紧急响应流程

1. 发现异常 → 立即通知核心团队
2. 确认攻击 → 触发 Emergency Stop
3. 止损 → 通过多签暂停合约
4. 事后分析 → 完整报告 + 修复方案
